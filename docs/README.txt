AI BUILDER SESSION 4: DATA ANALYSIS
====================================

FILES IN THIS BUNDLE
--------------------
college-data.csv
  200 US universities with 13 columns: admissions, tuition, earnings,
  completion rates, Nobel Prize counts, March Madness status, and more.
  Source: College Scorecard (US Dept of Education), Nobel Prize API,
  NCAA.

college_premium_by_state.csv
  State-level data: college vs. high school earnings, college premium,
  and cost of living index. Use this to enrich your analysis by merging
  it with the main dataset on the State column.
  Source: US Census, Bureau of Economic Analysis.

HOW TO USE
----------
1. Save this folder to your AI Builder directory
2. Open Claude Cowork and select your AI Builder folder
3. Start with college-data.csv and follow the session guide
4. Merge in college_premium_by_state.csv when you're ready to go deeper
